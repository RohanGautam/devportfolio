Projects

Blinkception

Our project enables users to interact with websites using just their facial movements. By using these movements, they can interact with web elements. Users are able to type text using a set pattern of facial gestures. These patterns are mapped to morse code.Additionally, our program employs facial recognition to remember previous users and is able to run using the pre-saved ratios of their facial features.Finally, we allow the users to send emergency Emails and WeChat messages. Made at iNTUition 2018.
